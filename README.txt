


The following steps will guide you to make a simple Login/logout functionality for your website.


File Descriptions
1. project.php (main page of this project incl. login validation for users and admin)
2. dashboard.php (Dashboard that will display dynamic information based on who is logged in w/ CRUD)
3. contactNew.php (Example file to show how guests infomation may be stored w/ C(Create Only))
4. db.php (holds the database informaton w/ Connection check)
5. logout.php (logs out the user/admin)
